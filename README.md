# La Abejería

Proyecto web creado con Vite + Vanilla JS para presentar el catálogo y contenido educativo de La Abejería.

## Requisitos
- Node.js 18+

## Instalación
```bash
npm install
```

## Desarrollo
```bash
npm run dev
```

## Build de producción
```bash
npm run build
```

## Vista previa de la build
```bash
npm run preview
```
